🎧 Spotify Analytics Dashboard (Power BI Project)

The project consists of an interactive Spotify Analytics Dashboard created using Power BI, focusing on the analysis of song and artist performance. The dashboard includes insights into various metrics, such as the total songs, unique artists, average popularity, and duration.

It includes visual analysis of explicit and non-explicit songs, the distribution of album types, and trends over time.

It shows the transformation of raw data from Spotify into valuable insights through the application of data visualization and business intelligence concepts.

🛠 Tools Used: Power BI, Figma
📊 Features: KPI Tracking, Trend Analysis, Interactive Dashboard, Data Visualization
